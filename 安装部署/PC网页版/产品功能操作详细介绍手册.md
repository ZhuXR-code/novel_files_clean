# 文件清理助手 · PC 网页版 产品功能操作详细介绍手册

网页版与 APP 端共用同一套产品能力，区别只在「形态」：网页版在浏览器里操作，左侧是「扫描配置」列表，主区域是文件 / 合集浏览。

## 1. 整体布局

- **左侧栏「扫描配置」**：管理多个扫描配置（文件夹、文件类型、递归、排除目录）。点配置即可开始一次扫描，结果形成一个「文库」。
- **主区域「文库内容」**：顶部有「列表 / 合集」切换，以及筛选、排序、搜索；底部是分页导航条。
- **右上角菜单**：合集模式下有「标记重复」「合集设置」。

## 2. 扫描配置

- 新增配置：名称（可选）、扫描文件夹、文件类型（如 `txt,md`）、是否递归、排除文件夹。
- 从配置一键「开始扫描 / 重新扫描」。扫描会递归读取文件、解析书名作者、计算内容指纹（SHA-256）用于识别重复。

## 3. 文库内容：列表模式

- 顶部「全部 / 已标记」筛选；按时间 / 名称 / 大小排序（点列头可切升 / 降序）。
- 关键字搜索框实时过滤。
- 每行可勾选（待删除）、加星标（标记）、点开查看明细。

## 4. 文库内容：合集模式

- 按「书名」把同名文件聚合成合集，便于发现重复。
- 「合集设置」可设：合集内文件数量区间（最少 ~ 最多，留空不限）、排除的合集书名。
- 合集可整体勾选 / 展开逐文件操作。

## 5. 分页导航条（大库浏览）

对齐 APP 端逻辑：

- **每页条数**：输入框（默认 100，范围 10~2000）填完 `onchange` 即生效（超出自动夹到 10~2000）。
- **跳至页**：输入框填目标页码，回车或点「跳转」直达。
- **首页 / 上一页 / 下一页 / 末页**：快速翻页。
- 页码信息显示「第 X / Y 页，共 N 条」。

> 这样面对十万级数据也只查一页，浏览流畅，不一次性拉全量。

## 6. 标记重复

合集模式下「标记重复」：按 PC 端逻辑（同作者 + 纯数字进度，保留最大进度与最大文件）智能勾选待删重复项；也可「按书名/作者相同标记」批量加星标、「清除标记」取消。

## 7. 选择并删除

勾选文件（或经「标记重复」自动勾选）后，底部出现「删除选中」；确认后逐条删除并实时显示进度与成功 / 失败数。删除通过系统授权进行，确认页可反选。

## 8. 设置

- **主题 / 字号**：调整界面外观（网页版跟随浏览器 / 系统）。
- **关键词替换（文件名数据清洗）**：维护「查找 → 替换」（留空=删除）规则，分扫描阶段（文件名）与解析阶段（书名/作者/进度/来源）作用，按顺序执行。
- **导出已标记清单** / **清空本地数据**：备份与重置。

## 9. 帮助

界面内「帮助」提供上述流程说明，随版本更新。
