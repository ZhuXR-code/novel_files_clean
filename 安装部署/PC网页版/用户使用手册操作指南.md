# 文件清理助手 · PC 网页版 用户使用手册操作指南

在浏览器里完成「扫描 → 整理 → 删除重复」。

## 准备

1. 部署人员已按《安装部署手册》把服务跑起来，并告诉你访问地址（如 `http://localhost:8000`）。
2. 准备一个放着 txt/md 文件的文件夹。

## 步骤一：新建扫描配置并开始

1. 左侧「扫描配置」点 **+ 新增**。
2. 填写：
   - 名称（可选）；
   - 扫描文件夹（选你的目标目录）；
   - 文件类型（默认 `txt`，可 `txt,md`）；
   - 是否递归子目录；排除文件夹（可选）。
3. 保存后点 **开始扫描**，等待完成。

> 扫描只在你指定的文件夹内进行，数据落在部署的 MySQL，不上传公网（取决于部署方式）。

## 步骤二：浏览文库

1. 扫描完成进入主区域。顶部切 **列表 / 合集**：
   - 列表：所有文件平铺，可「全部/已标记」筛选、排序、搜索。
   - 合集：按书名聚合，相同书名归一组。

## 步骤三：大库分页浏览

数据很多时，用底部「分页导航条」：

- 想每页看 200 条：在「每页」框输入 `200`（自动生效，范围 10~2000）。
- 想到第 50 页：在「跳至页」框输入 `50` 回车或点「跳转」。
- 也可用 **首页 / 上一页 / 下一页 / 末页** 翻页。

## 步骤四：找出重复（合集模式）

1. 顶部切到 **合集**。
2. 右上角菜单 **合集设置**：比如只想看「组内 ≥ 2 个文件」的合集就填「最少 2」；想排除某些书名填到「排除的合集」。点确认应用。
3. 右上角菜单 **标记重复**：自动勾选「同作者 + 纯数字进度、保留最大进度与最大文件」的待删重复项。
4. 也可手动勾选 / 加星标。

## 步骤五：删除重复

1. 选好后底部出现 **删除选中**（显示已选数量）。
2. 点它进入确认页，可逐项取消不想删的。
3. 点 **确认删除**，逐条删除并实时显示进度与成功 / 失败数。

> 删除前建议先到「设置 → 导出已标记清单」备份待删列表。

## 步骤六：其他实用功能

- **关键词替换（文件名清洗）**：设置里维护「查找 → 替换」规则，扫描 / 解析阶段自动清洗书名、作者等。
- **导出已标记清单 / 清空本地数据**：备份与重置。
- **帮助**：随时查看说明。
