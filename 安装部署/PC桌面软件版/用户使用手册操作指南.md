# 文件清理助手 · PC 桌面软件版 用户使用手册操作指南

在一个独立的软件窗口里完成「扫描 → 整理 → 删除重复」，无需浏览器、无需联网。

## 准备

1. 已安装并能打开「文件扫描管理系统（本地版）」（见《安装部署手册》）。
2. 准备一个放着 txt/md 文件的文件夹。

## 步骤一：新建扫描配置并开始

1. 左侧「扫描配置」点 **+ 新增**。
2. 填写：
   - 名称（可选）；
   - 扫描文件夹（选你的目标目录）；
   - 文件类型（默认 `txt`，可 `txt,md`）；
   - 是否递归子目录；排除文件夹（可选）。
3. 保存后点 **开始扫描**，等待完成。

> 桌面版所有数据都存在**本机 SQLite 数据库**（程序安装目录内 `FileScannerData\file_scanner.db`），不上传、不联网，隐私安全。

## 步骤二：浏览文库

扫描完成进入主区域。顶部切 **列表 / 合集**：

- 列表：所有文件平铺，可「全部/已标记」筛选、排序、搜索。
- 合集：按书名聚合，相同书名归一组。

## 步骤三：大库分页浏览

数据很多时，用底部「分页导航条」：

- 想每页看 200 条：在「每页」框输入 `200`（自动生效，范围 10~2000）。
- 想到第 50 页：在「跳至页」框输入 `50` 回车或点「跳转」。
- 也可用 **首页 / 上一页 / 下一页 / 末页** 翻页。

## 步骤四：找出重复（合集模式）

1. 顶部切到 **合集**。
2. 右上角菜单 **合集设置**：只想看「组内 ≥ 2 个文件」的合集就填「最少 2」；想排除某些书名填到「排除的合集」。点确认应用。
3. 右上角菜单 **勾选重复**：按「同一合集内、以 (作者+小说名) 子分组」套用五则规则自动勾选待删重复项（① 文件名+大小+小说名+作者+进度 完全相等则非最新全勾选；② 进度全为纯数字则进度最大者不勾选、其余纯数字全勾选；③ 进度含中文不勾选，但若同组有带『完结』等文件名且进度最大文件更小，则进度最大文件也勾选；④ 本组内文件大小最大者不勾选；⑤ 进度为「完结+数字番外」组合时按数字排序，数字最大者不删、其余删（被删者若恰为本组文件大小最大者也不删））。
4. 也可手动勾选 / 加星标。

> 五则规则的完整判定表与具体举例（完全相等去重、纯数字进度对比、含中文进度与完结特例、最大文件不勾选原则等），详见本目录《产品功能操作详细介绍手册》第 6 节。

## 步骤五：删除重复

1. 选好后底部出现 **删除选中**（显示已选数量）。
2. 点它进入确认页，可逐项取消不想删的。
3. 点 **确认删除**，逐条删除并实时显示进度与成功 / 失败数。

> 桌面版删除的是**你本机磁盘上的真实文件**，删除前建议先到「设置 → 导出已标记清单」备份待删列表。

## 步骤六：其他实用功能

- **关键词替换（文件名清洗）**：设置里维护「查找 → 替换」规则，扫描 / 解析阶段自动清洗书名、作者等。
- **操作日志查看**：软件内「调试日志」面板可切换「操作日志 / 调试日志」，随时回看勾选重复、删除、清空等操作的留痕，并一键复制。
- **导出已标记清单 / 清空本地数据**：备份与重置（清空只清 SQLite 里的记录，不动磁盘文件）。
- **帮助**：随时查看说明。

## 关闭软件

直接点窗口右上角关闭按钮即可，程序会自动停掉后台服务并完整退出。数据已实时保存在本机数据库，下次打开自动加载。

## 数据迁移 / 换机

想把数据带到另一台电脑：

- 若用**便携模式**（`FileScanner.exe --portable`）：直接把整个程序目录（含 `FileScannerData\`）拷走即可。
- 若用默认模式：拷贝 `%APPDATA%\FileScanner\file_scanner.db` 到新机同名目录。
