# 文件清理助手 · APP 端（Android）用户使用手册操作指南

面向最终用户，跟着做即可完成「扫描 → 整理 → 删除重复」。

## 准备

1. 安装后打开「文件清理助手」。
2. 准备一个放着 txt/md 小说 / 文本文件的文件夹。

## 步骤一：开始一次扫描

1. 首页点 **开始扫描** → 进入「扫描配置」列表。
2. 点右下角 **+** 新增配置：
   - 名称（可选，便于区分）；
   - 点 **选择文件夹**，在系统文件夹选择器里选中你的目标文件夹（建议勾选「始终允许访问」，否则下次还要授权）；
   - 文件类型默认 `txt`，可改成 `txt,md`；
   - 需要连子目录就保持「递归」开启；想跳过某些目录填「排除的文件夹」。
3. 保存配置后点 **开始扫描**，进入「扫描进度」页实时查看进度。扫描进行中可随时点 **停止扫描**；停止成功（或扫描自然结束）后，页面会显示 **重新扫描** 按钮，一键用同一配置再次扫描，无需重新选择文件夹。

> 扫描只会读取你授权的文件夹，不会上传任何文件。

## 步骤二：浏览文库

1. 回到首页点 **查看文库**，或扫描完成直接「查看结果」。
2. 点某个文库进入内容页。
3. 顶部可在 **列表 / 合集** 间切换：
   - 列表：所有文件平铺，可用「全部 / 已标记」筛选、排序、搜索。
   - 合集：按书名聚合，相同书名归一组。

## 步骤三：大库分页浏览

数据很多（几万条）时，用底部「分页导航条」：

- 想每页看 200 条：在「每页」框输入 `200`，点 **应用**。
- 想到第 50 页：在「跳到」框输入 `50`，点 **跳转**。
- 也可用 **首页 / 上一页 / 下一页 / 末页** 翻页。

列表太长时点右下角「回到底层 / 回到顶层」可快速滚动。

## 步骤四：找出重复（合集模式）

1. 顶部切到 **合集**。
2. 点右上角 **⋮** → **合集设置**：
   - 比如只想看「组内 ≥ 2 个文件」的合集，就填「最少 2」；
   - 想排除某些书名，填到「排除的合集」。
   - 点 **确认** 应用。
3. 再点右上角 **⋮** → **勾选重复**：应用按「同一文库内、以 (作者+小说名) 子分组」套用五则规则自动勾选待删重复项（① 文件名+大小+小说名+作者+进度 完全相等则非最新全勾选；② 进度全为纯数字则进度最大者不勾选、其余纯数字全勾选；③ 进度含中文不勾选，但若同组有带『完结』等文件名且进度最大文件更小，则进度最大文件也勾选；④ 本组内文件大小最大者不勾选；⑤ 进度为「完结+数字番外」组合时按数字排序，数字最大者不删、其余删（被删者若恰为本组文件大小最大者也不删））。算法与 PC 端 backend/dup_logic.py 完全一致。
4. 也可手动点星标、勾选框来增删选择。

> 五则规则的完整判定表与具体举例（完全相等去重、纯数字进度对比、含中文进度与完结特例、最大文件不勾选原则等），详见本目录《产品功能操作详细介绍手册》第 6 节。

## 步骤五：删除重复

1. 底部有 **清除勾选** 和 **删除** 两个按钮：点 **清除勾选** 清空所有勾选；点 **删除** 进入删除确认页（显示已选数量）。
2. 在删除确认页，可逐项取消不想删的。
3. 点 **确认删除**，前台服务逐个删除并实时显示进度与成功 / 失败数。

> 删除前建议先到「设置 → 导出已标记文件清单」备份待删列表；删除通过系统授权进行，确认页仍可反选。

## 步骤六：其他实用功能

- **关键词替换（文件名清洗）**：设置里维护「查找 → 替换」规则，扫描 / 解析阶段自动清洗书名、作者等。
- **调试日志**：设置里点「调试日志」可查看、刷新、复制、清空应用运行日志，删除 / 扫描等关键操作都有留痕，出问题时方便贴给开发者。
- **阅读设置**：设置里可切换预览页滚动条为「竖向」或「横向（顶部）」，默认竖向。
- **主题 / 字号**：设置里切换深色、调大字体，适配你的手机。
- **清空本地数据**：想彻底重来时，在设置里清空（不可恢复）。
- **帮助**：随时点首页右上角 **?** 查看说明。
